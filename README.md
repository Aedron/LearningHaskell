
## Haskell Learning

### GHC
* __编译__:  
输入 `ghc --make ./helloworld`  
编译 `helloworld.hs` 到 `helloworld` 二进制文件  


### GHCI
* __进入交互模式__:  
输入 `ghci`  
然后在 ghci 中输入: `:l <FileName>`  

* __重新载入__:  
修改文件后使用 `:r` 重新载入    

* __查看类型__:  
`:t <Any>`
